How to install:
1. Download and install a server solution stack e.g. (WAMP, XAMPP )
2. After installing, copy the application package into the www directory in the localhost.
3. Open the phpmyadmin in the browser thru the url http://localhost/phpmyadmin
4. Import the file hotel_dabaw.sql as this will serve as the program's database. 
5. The program opens through the url http://localhost/hotel_dabaw/public/admin.php
6. The client-side booking web application opens through the url http://localhost/hotel_dabaw/public/index.php
7. In the admin program, the default username is superadmin and the default password is also superadmin. You can now navigate through the pages. Enjoy!
